# <center> Frontend

### Core technológiák
- **Keretrendszer:** [Vue.js](https://vuejs.org/)
- **Állapotkezelés:** [Pinia](https://pinia.vuejs.org/)
- **Routing:** [Vue Router](https://router.vuejs.org/)

### UI és megjelenés
- **Design rendszer:** [Bootstrap](https://getbootstrap.com/)
- **Ikonok:** [Font Awesome](https://fontawesome.com/), [flag-icons](https://flagicons.lipis.dev/)

### Egyéb funkciók
- **HTTP kliens:** [Axios](https://axios-http.com/)
- **Többnyelvűség (nyelvi támogatás):** [Vue I18n](https://vue-i18n.intlify.dev/)
