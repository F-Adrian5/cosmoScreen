# <center> Futtatási útmutató

**A projekt futtatásához két terminál szükséges: egy a frontendhez és egy a backendhez.**

## 1. Backend indítása

  ```bash
  cd backend
  npm run dev
  ```

## 2. Frontend indítása

  **A projekt gyökérmappájában:**

  ``` bash
  npm run dev
  ```

## 3. Alkalmazás megnyitása
 - **A frontend a terminálban megjelenő címen érhető el (pl. http://localhost:5173).**
